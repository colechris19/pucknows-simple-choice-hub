## About pucKnows

pucKnows is a compact decision making tool that operates entirely in the browser, offering a simple way to work through options in a structured format. Because it is implemented with HTML and presented as a web app, it can be opened in modern browsers without requiring a separate desktop install.

The project is aimed at use cases where a clear decision flow matters more than a heavy application framework. It is a practical fit for personal planning, small group evaluation, or any process where you want one place to review alternatives and arrive at a choice.
